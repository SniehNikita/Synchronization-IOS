This program creates molecule of H2O using H and O atoms.
It creates a process for each atom and the syncronizes them.


Usage:
./proj2 int1 int2 int3 int4

4 required params:
    int1 - number of oxygen atoms
    int2 - number of hydrogen atoms
    int3 - time to create 1 atom
    int4 - time to create 1 molecule
